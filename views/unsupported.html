Unsupported Browser

It appears that your web browser is not supported by Calagora. Luckily, there are several free browsers available on the web with which you will be able to use Calagora!

Most newer versions of Google Chrome, Mozilla Firefox, Microsoft Edge, and Apple Safari are supported. If you have an older version of one of these browsers, we recommend upgrading to the newest version of it. If you do not use one of those browsers, we recommend switching to one.

For all mobile phones and desktop PCs, we recommend Google Chrome.

We apologize for the inconvenience. If you believe that your browser is a newer browser and should be supported, please send us an email at support@calagora.com with information about the browser you are currently using. We will do our best to extend support to that browser.
